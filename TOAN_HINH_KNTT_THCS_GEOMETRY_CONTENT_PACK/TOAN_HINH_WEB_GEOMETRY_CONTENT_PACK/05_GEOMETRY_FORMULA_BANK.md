# 05 — Formula Bank Toán hình THCS

> Dùng làm ngân hàng công thức để Antigravity tham chiếu khi tạo bài tập. Công thức vẫn cần giáo viên duyệt theo bài/chương đang triển khai.

## Lớp 6

| Chủ đề | Công thức/tính chất |
|---|---|
| Hình vuông | Chu vi `$P = 4a$`, diện tích `$S = a^2$`. |
| Hình chữ nhật | `$P = 2(a+b)$`, `$S = ab$`. |
| Tam giác đều | Chu vi `$P = 3a$`; các cạnh bằng nhau, các góc bằng nhau. |
| Lục giác đều | Chu vi `$P = 6a$`; có thể chia thành 6 tam giác đều bằng nhau. |
| Hình bình hành | `$S = a \cdot h$` với `$h$` là chiều cao ứng với đáy `$a$`. |
| Hình thoi | Chu vi `$P = 4a$`; diện tích thường dùng `$S = \frac{d_1 d_2}{2}$` nếu biết hai đường chéo. |
| Hình thang | `$S = \frac{(a+b)h}{2}$` với `$a,b$` là hai đáy. |
| Trung điểm | M là trung điểm AB khi M nằm giữa A,B và `$MA = MB$`. |
| Góc | Góc nhọn `<90°`, vuông `90°`, tù `>90°` và `<180°`, bẹt `180°`. |

## Lớp 7

| Chủ đề | Công thức/tính chất |
|---|---|
| Góc kề bù | Tổng hai góc kề bù bằng `$180°$`. |
| Góc đối đỉnh | Hai góc đối đỉnh bằng nhau. |
| Tia phân giác | Chia một góc thành hai góc bằng nhau. |
| Tam giác | Tổng ba góc trong tam giác bằng `$180°$`. |
| Tam giác bằng nhau | Các trường hợp: c-c-c, c-g-c, g-c-g; tam giác vuông có các trường hợp riêng. |
| Tam giác cân | Hai cạnh bên bằng nhau; hai góc ở đáy bằng nhau. |
| Bất đẳng thức tam giác | Tổng hai cạnh bất kỳ lớn hơn cạnh còn lại. |
| Quan hệ góc-cạnh | Cạnh lớn hơn đối diện góc lớn hơn và ngược lại. |
| Hình hộp chữ nhật | `$V = abc$`, diện tích toàn phần `$2(ab+bc+ca)$`. |
| Lập phương | `$V=a^3$`, `$S_{tp}=6a^2$`. |
| Lăng trụ đứng | `$V = S_{đáy}\cdot h$`, `$S_{xq}=p_{đáy}\cdot h$`. |

## Lớp 8

| Chủ đề | Công thức/tính chất |
|---|---|
| Tứ giác | Tổng bốn góc bằng `$360°$`. |
| Hình bình hành | Cạnh đối song song/bằng nhau; góc đối bằng nhau; đường chéo cắt nhau tại trung điểm. |
| Hình chữ nhật | Hình bình hành có một góc vuông; đường chéo bằng nhau. |
| Hình thoi | Bốn cạnh bằng nhau; đường chéo vuông góc và phân giác góc. |
| Hình vuông | Vừa là hình chữ nhật vừa là hình thoi. |
| Thalès | Đường thẳng song song với một cạnh của tam giác tạo ra các đoạn tương ứng tỉ lệ. |
| Đường trung bình tam giác | Song song với cạnh thứ ba và bằng nửa cạnh ấy. |
| Phân giác tam giác | Chia cạnh đối diện thành hai đoạn tỉ lệ với hai cạnh kề. |
| Tam giác đồng dạng | Góc tương ứng bằng nhau, cạnh tương ứng tỉ lệ. |
| Pythagore | Tam giác vuông: `$a^2+b^2=c^2$` với `$c$` là cạnh huyền. |
| Hình chóp đều | Diện tích xung quanh bằng tổng diện tích các mặt bên; thể tích nếu có dùng: `$V=\frac13 S_{đáy}h$`. |

## Lớp 9

| Chủ đề | Công thức/tính chất |
|---|---|
| Tỉ số lượng giác | `sin = đối/huyền`, `cos = kề/huyền`, `tan = đối/kề`, `cot = kề/đối`. |
| Đường tròn | Đường kính `$D=2R$`; dây lớn nhất là đường kính. |
| Độ dài cung | `$l = \frac{n}{360}\cdot 2\pi R$`. |
| Diện tích quạt | `$S = \frac{n}{360}\cdot \pi R^2$`. |
| Vành khuyên | `$S = \pi(R^2-r^2)$`. |
| Tiếp tuyến | Tiếp tuyến vuông góc bán kính tại tiếp điểm. |
| Góc nội tiếp | Số đo góc nội tiếp bằng nửa số đo cung bị chắn. |
| Tứ giác nội tiếp | Hai góc đối bù nhau. |
| Đa giác đều | Góc ở tâm `$=360°/n$`; góc ngoài `$=360°/n$`; góc trong `$=180°-360°/n$`. |
| Hình trụ | `$S_{xq}=2\pi Rh$`, `$S_{tp}=2\pi Rh+2\pi R^2$`, `$V=\pi R^2h$`. |
| Hình nón | `$S_{xq}=\pi Rl$`, `$S_{tp}=\pi Rl+\pi R^2$`, `$V=\frac13\pi R^2h$`. |
| Hình cầu | `$S=4\pi R^2$`, `$V=\frac43\pi R^3$`. |
