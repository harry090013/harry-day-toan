# Toán Hình Web — Geometry Content Pack KNTT THCS 6–9

Bộ tài liệu này dùng để đưa cho Antigravity đọc trước khi tạo dữ liệu **Toán hình** cho website nội bộ dạy học sinh mất gốc.

## Mục tiêu

- Lấp đầy nội dung Toán hình lớp 6–9 theo bộ **Kết nối tri thức**.
- Không tự thêm bài học ngoài curriculum đã có.
- Mỗi bài có: lý thuyết tóm tắt → ví dụ mẫu → dạng toán nhỏ → trắc nghiệm sinh ngẫu nhiên → lời giải chi tiết hiện ngay.
- Ưu tiên học sinh mất gốc: đọc hình, nhận dạng dạng toán, áp dụng công thức/tính chất, tránh lỗi sai.

## Cách dùng nhanh

1. Giải nén thư mục này vào project, ví dụ: `docs/toan-hinh-content-pack/`.
2. Mở Antigravity ở root project.
3. Chạy lệnh liệt kê file để Antigravity thấy toàn bộ tài liệu.
4. Copy nội dung trong `00_ANTIGRAVITY_MASTER_PROMPT.md` vào Antigravity.
5. Yêu cầu làm từng chương, bắt đầu từ lớp 6 hoặc chương bạn đang dạy.
